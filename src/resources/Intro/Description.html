Welcome to PPA4C!

 

PPA4C is a program to practice using pseudocode (informal language describing a program in English) to write C programs. The exercises included with PPA4C use an easy multiple-choice format and include reference background material for each topic to help study or review various topics in C programming.

This short exercise will show you how to use the program.

On the left is the Topics list: use this to navigate to exercises and different categories. The arrows < and > on the window divider can be used to hide and show the topics list (you can also drag the divider around).

Once an exercise is chosen, it appears to the right of the divider.

The description tab (where you are now) will normally show the description for an exercise's topic, giving background information needed to complete the exercise.

After reading the description, go to the Practice tab at the top; you can go between the description and practice tabs as needed. Go over to the Practice tab to learn more about it.
