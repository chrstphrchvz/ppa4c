printf() part 2

Description

Here's where the formatting part of printf() is useful: besides just printing strings, printf() can also print data from a program.

The string given to printf() can contain conversion specifiers (a.k.a. format placeholders) which tell printf() to replace that specifier with data when it is run.

A few conversion specifiers are:

%d

Decimal (base 10) integer

%x

Hexadecimal (base 16) integer

%f

Single precision floating point

%lf

Double precision floating point

%c

Character

%s

Null-terminated string

More conversion specifiers are available as well as formatting parameters for each that give a higher level of control over the output. The useful thing about printf() is its ability to convert a variety of things in memory (e.g. 0xFFFFFFDD or 0x576F7700) to something useful (-35 or "Wow") so that we don't have to ourselves, and we can go on to write useful programs!

Syntax

Be sure to have the line

#include <stdio.h>

at the beginning of the .c file for a program.

The first argument is the string with any conversion specifiers; for each specifier used, there must be an argument corresponding to the data to be printed, in the order they appear in the string.

Code example

printf("%s, %s %d, %d\n", "Friday", "February", 13, 2015);

printf("%d:%d%cM", 11, 10, 'A');

prints

Friday, February 13, 2015

11:10AM

The arguments may be either literals (as above) or names of variables:

int a = 49;

int b = 30;

int unknown = a * b;

printf("%d * %d = %d", a, b, unknown);

prints

49 * 30 = 1470
