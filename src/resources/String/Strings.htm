Description

A String is a one dimensional array of Characters followed by a null character '\0'

It functions similar to an Array but has a lot of uses to programmers because it allows us to form words and sentences using characters rather than only using one character at a time.

 

A String is declared in the same way as an array only with the variable type char.

 

There are a lot of functions for programming that are made specifically for the use of strings.

 

String length

                num = strlen(String);

Now num is equal to the length of the string.

 

Concatenate

                char string1[50] = "My Name is ";

                char string2[4] = "Bob";

                strcat(string1, string2);

This essentially merges the two strings together. string1 is now "My Name is Bob"

 

Copy

                strcpy(string1, string2);

This function copies string2 into string1.

 

Compare

                num=strcmp(string1, string2);

This function compares the two strings and returns 0 if the two strings are the same.
