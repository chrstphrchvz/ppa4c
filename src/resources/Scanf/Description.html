Scanf()

SCANF

Introduction

The function, scanf(), reads data from the console or from a FILE stream, parses it, and stores the results away in variables you provide in the argument list. scanf() has a prerequisite that must be met prior to being called. Firstly, there must some variable that can store whatever information is about to be collected from scanf().

Percent-Codes

scanf() uses a variety percent-codes to denote what type of data will collected and stored.

%d Reads an integer to be stored in an int. The integer may be signed.

%f Reads a floating point number. The data will be stored in a float.

%s Reads a string. Please note that scanf() will stop at the first instance of whitespace character. With %s, a programmer can also designate a specified field width. (e.g. %5s)

%c Reads a character that will be stored in a char. The programmar may also specify the field width.



Example

//---------------------------------------------------------------------------------------------------

int a; // Declared variable where data will be stored.

scanf("%d", &a); // Store an int into variable a

//---------------------------------------------------------------------------------------------------

float d; //declare a float variable

scanf( "%", &d ); //Store a float into a variable d

//---------------------------------------------------------------------------------------------------

char s[50]; //Declare a char array capable of holding 50 characters

scanf( "%10s", s ); //Store a string of field width 10 into s

//---------------------------------------------------------------------------------------------------
