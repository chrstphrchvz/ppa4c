Insertion Sort

 

 

Insertion Sort does exactly as it sounds; it inserts each element of the array into its proper position. Insertion sort goes through the array and iterates down the array for each element to fix into place just like this image bellow.

 

Coding the insertion is fairly strait forward, please refer too Bubble sort and its example, an algorithm that is typically laid out for insertions is this:

 

for (k = i; k > 1 and a[k] < a[k-1]; k--)

        swap a[k,k-1]

 

Though in the problem it will be slightly coded differently keep in mind that it is similar to bubble sort besides this algorithm.
