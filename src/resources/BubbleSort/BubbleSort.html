Bubble Sort

 

 

Bubble sort is a sorting algorithm, the way it works is that it passes through a list comparing and swaps anything in the wrong order. It keeps repeating from the beginning of the list until there are no more comparisons to be made.

 

 

 

This image shows a quick example of how bubble sort works. We first must go over an area which is simply a field of elements. For example

int array[1000];

 

An array could vary in size, but by using a bubble sort you can sort the field’s components and order them. Depending on what order, you can compare each component/value and swap them to your order. Bubble sort is the least efficient of the 3 sorts that we are doing but it is a good concept to know. Here is an example of bubble sort:

 

/* Bubble sort code */

 

#include <stdio.h>

 

int main()

{

  int array[100], n, c, d, swap;

 

  printf("Enter number of elements\n");

  scanf("%d", &n);

 

  printf("Enter %d integers\n", n);

 

  for (c = 0; c < n; c++)

    scanf("%d", &array[c]);

 

  for (c = 0 ; c < ( n - 1 ); c++)

  {

    for (d = 0 ; d < n - c - 1; d++)

    {

      if (array[d] > array[d+1]) /* For decreasing order use < */

      {

        swap       = array[d];

        array[d]   = array[d+1];

        array[d+1] = swap;

      }

    }

  }

 

  printf("Sorted list in ascending order:\n");

 

  for ( c = 0 ; c < n ; c++ )

     printf("%d\n", array[c]);

 

  return 0;

}
