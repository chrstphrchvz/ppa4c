printf() PART 1

Description

The printf() function, short for print formatted, is one of the many functions used to output a variety of data to stdout for the user to read.

printf() takes a string as its first argument (for now, a string is a sequence of characters between double quotes); this string can just contain a constant/hard-coded message, like "Hello World!" , but it can also be used to print data that might be generated by the program, such as a variable (cf. part 2).

Syntax

For a program to use printf() , it needs the standard I/O library, stdio.h ; this is done by placing the line

  #include <stdio.h>

at the beginning of the .c file for a program.

printf() is called in its own statement--no declarations, operations, etc. are needed:

  printf();

is how it would appear on its own line.

Code Example

To print a single string with printf(), place the string inside the parentheses:

  printf("Hello World!");
