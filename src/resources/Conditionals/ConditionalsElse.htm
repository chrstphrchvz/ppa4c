Conditionals (else) 

               It is possible for multiple branches to exist, creating complex trees. A variation of the if statement exist, called the if...else statement. The else statement MUST  follow an if block, and it only executes if the preceding if clause is false. While the two statements are similar in that multiple branches are available, the main difference between the two is that in the if...else statement, there is only one exit to the control structure.

EXAMPLE USAGE:

int x = 2;

if(x%2 == 0 )

                printf("x is even");

else

                printf("x is odd");

 

outputs the following

x is even
