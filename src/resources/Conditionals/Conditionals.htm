Conditionals - Introduction

                Conditional statements in C is a construct that allows the programmer to continue program flow of control between two or more choices. The diagram below illustrates the flow of control for a basic two-way decision (if statement).

                It is possible for multiple branches to exist, creating complex trees. A variation of the if statement exist, called the if...else statement. While the two statements are similar in that multiple branches are available, the main difference between the two is that in the if...else statement, there is only one exit to the control structure.

                The final method of selection is the switch statement, which enables multilevel selection.

 

 

 

EXAMPLE USAGE

IF:

if (a > b)

                return a;

returns a only if it is greater than, but not equal to, b.

IF ELSE:

int x = 2;

if(x%2 == 0 )

                printf("x is even");

else

                printf("x is odd");

 

outputs the following

x is even

 

SWITCH:

int x = 1;

switch(x)

{

                case 1: printf("This is the number 1"); break;

                case 2: printf("This is the number 2"); break;

}

outputs the following

This is the number 1
