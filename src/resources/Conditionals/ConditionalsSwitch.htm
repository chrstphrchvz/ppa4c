CONDITIONALS - SWITCH

Description

The final method of selection is the switch statement, which enables multiway selection.


While it is possible to use multiple if statements to enable multiway selection, an easier way of doing so is by using the switch statement.

The value in Expression is a single variable type, and program flow of control is determined based on case.

The branches in a switch statement are usually mutually exclusive. Additionally, a default case is available in case none of the cases are true.

Syntax

The syntax begins with a switch statement and its expression followed by the body containg multiple cases:

switch(/*expression*/)

{

/*Multiple case statements*/: /*Multiple break statements*/;

/*Optional default statements*/: /*default expression*/;

}

Code Example

int x = 1;

switch(x)

{

                case 1: printf("This is the number 1"); break;

                case 2: printf("This is the number 2"); break;

            default: printf("That is not a number!");break;

}

outputs the following

This is the number 1
