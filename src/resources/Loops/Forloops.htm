For Loops 

Description

For loops will execute a statement for specific number of iterations

Syntax

for(variable=number;condition;variable modification)
{
  statement;
}

Code Example:

So for example you want to multiply a number y by 5, 10 times.
You would want to do the following

-> while x<10
--> multiply y by 5
--> add 1 to x

Here is the code for that for loop

y=1;
for(x=0;x<10;x=x+1)
{
  y=y*5;
}

This is what will happen

- The diagram above shows that x will be compared to 10.
- The contents of variable x will be set to 0.
- Since 0 is less than 10 y will be multiplied by 5. This will set y = 5.
- The x will than be increased by 1 as specified in the FOR statement.
- In the next iteration x will be equal to 1.
- Since x will be less than 10, y will again be multiplied by 5.
- At this point y will be equal to 25.

This pattern will continue until x is equal to or less than 10.
