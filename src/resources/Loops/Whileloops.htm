While Loops 

Description

While loops While loops will execute a statement while a condition is true.

Syntax

while(condition) { statement }

Code Example:

So for example you would like to print hello 25 times.

To do this you will want to do the following;

-> While x < 25
--> print hello
--> add 1 to x

Here is the code for that while loop

x=0;
while(x < 25)
{
  printf("hello");
  x=x+1;
}

This is what will happen

- The diagram above shows that x will be compared to 25.
- In the first iteration x will be equal to 0.
- Since the content of x is less than 25 hello will be printed and x will be increased by one.
- In the second iteration x will be equal to 1. This time it is also less than 25.

This pattern continues until x equal to or above 25. At this point the loop will end.
