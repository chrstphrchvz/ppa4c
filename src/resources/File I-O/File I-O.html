File I/O

Description and Introduction:

In this session you will learn how to manipulate and create files through C. Main functions you will note are: fopen, fwrite, fread, fprintf, fscanf, fgetc, fputc.

First you must access a file with a pointer like this example:

FILE *someFile;

Then you must open the file to manipulate it (or to create a file) you must set a location and its mode:

FILE *fopen(const char *filename, const char *mode);

There are 6 different modes that allowed:

r – open for reading

w- open for writing (file need not exist)

a – open for appending (file need not exist)

r+ - open for reading and writing, start at beginning

w+ - open for reading and writing (overwrite file)

a+ - open for reading and writing (append if file exists)

Keep in mind for future coding, depending on location of the file you are accessing you may need to pin point by adding a “..//” to find it. So otherwise it may look something like this:

FILE *someFILE;

someFILE = fopen(..\\YO.mama.txt”. “r”);

Also note you MUST ALWAYS close a file near the end of your main. So right before your return(0);  put something like this:

fclose(something);

Now for using the rest of functions that we will go over. fprintf and fscanf allows you to output to a file and input from the user.

Syntax for fprintf:                           fprintf(something, “Testing… \n”);

fgets “gets” data from a file an example is:                      int fgetc (FILE *somefile);

and fputc function allows you to put in data in a file

int fputc ( int c, FILE *fp );
